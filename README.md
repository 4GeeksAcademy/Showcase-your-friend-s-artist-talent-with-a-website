# Franchesco Stabile | IA, Data y Desarrollo Web

Sitio web estático creado como proyecto de portfolio para 4Geeks Academy. Presenta el perfil profesional de Franchesco Stabile, conectando experiencia industrial, desarrollo web, analítica de datos, Power BI, machine learning y automatización con inteligencia artificial.

## Vista General

Este proyecto funciona como una landing page personal y profesional. El objetivo es mostrar una marca técnica clara, una narrativa de transición hacia tecnología y una estructura visual lista para compartir como parte de un portfolio junior.

## Características

- Diseño responsive para escritorio y móvil.
- Secciones de presentación, trayectoria y proyectos.
- Imágenes optimizadas para reforzar la identidad visual.
- Estructura simple, fácil de mantener y preparada para GitHub Pages.
- Enfoque profesional orientado a IA Engineering, Data Analytics y desarrollo web.

## Cómo Ejecutarlo

Puedes abrir `index.html` directamente en el navegador o levantar un servidor local:

```bash
python3 -m http.server 8000
```

Después entra en:

```text
http://localhost:8000
```

## Objetivo Del Proyecto

Crear una página personal que funcione como carta de presentación para oportunidades junior en tecnología, destacando habilidades prácticas en desarrollo web, datos e inteligencia artificial.

## Notas De Privacidad

- `assets/images/` contiene únicamente imágenes finales usadas por la web.
- `docs/checks/` conserva capturas de revisión visual.
- El material privado, archivos fuente originales o notas personales deben quedarse fuera de Git, por ejemplo en `.private/`.
