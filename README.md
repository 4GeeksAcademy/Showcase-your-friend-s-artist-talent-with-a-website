# Showcase your friend's artist talent with a website

Sitio web estático para presentar el perfil, trayectoria y proyectos de Franchesco Stabile.

## Estructura

```text
.
├── index.html
├── css/
│   └── styles.css
├── assets/
│   └── images/
│       ├── franchesco-hero.jpg
│       └── franchesco-profile.jpg
└── docs/
    ├── checks/
    └── source-material/
```

## Cómo abrirlo

Abre `index.html` directamente en el navegador o levanta un servidor local:

```bash
python3 -m http.server 8000
```

Después entra en `http://localhost:8000`.

## Notas

- `assets/images/` contiene las imágenes finales usadas por la web.
- `docs/checks/` guarda capturas de revisión visual.
- `docs/source-material/` conserva material de referencia del proyecto.
